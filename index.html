Title Matching game

Created a tile matching game using Java.

On the screen is a grid where and the colors appear randomly. The players have to click on the grid that has more than 2 tiles of same color that are adjacent to each other to clear the grid. The game is over once the grids are completely filled with color.

Cafe Inventory

Created a store inventory in Java. Since it is an inventory, you can add, list, and delete new food items. If you want to know how many items remaining, use the list function to get the amount of specified items left. Then, when a consumer arrives, depending of what and how much they bought, adjust the number of items and then calculate the total amount earned. Each item has a name and cost.
